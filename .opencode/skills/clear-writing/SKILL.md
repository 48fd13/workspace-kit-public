# Clear Writing

Improve writing so it is direct, useful, and easy to act on.

## Rules

- Prefer concrete nouns and verbs.
- Remove filler, throat-clearing, and performative emphasis.
- Preserve technical accuracy and nuance.
- Use bullets for scannable procedures.
- Use tables only when comparing or routing.
- Separate facts, assumptions, risks, and next actions.
- Do not delete important context for the sake of brevity.
