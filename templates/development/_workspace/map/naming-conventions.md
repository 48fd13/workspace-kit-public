# Naming Conventions

Use durable artifact names:

```text
YYYY-MM-DD-topic-stage-vNN.md
```

Stages: `plan`, `validation`, `review`, `handoff`, `decision`.

Common artifact locations:

```text
_workspace/outputs/handoffs/YYYY-MM-DD-topic-handoff-vNN.md
_workspace/outputs/plans/YYYY-MM-DD-topic-plan-vNN.md
_workspace/outputs/reviews/YYYY-MM-DD-topic-review-vNN.md
_workspace/outputs/validations/YYYY-MM-DD-topic-validation-vNN.md
_workspace/backlog/items/YYYY-MM-DD-short-slug.md
_workspace/runs/active/YYYY-MM-DD-topic-slug/
_workspace/runs/archive/YYYY-MM-DD-topic-slug/
```
