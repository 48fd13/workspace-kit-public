# Workspace

Work-oriented folder-workflow workspace.

- `map/`: routing, naming, workspace map
- `workflows/`: code, docs, review, handoff workflows
- `runbooks/`: validation and git procedures
- `context/reference/`: template-provided engineering and documentation context
- `context/project/`: project-specific context discovered after initialization
- `context/reference/folder-workflow-system-primer.md`: local agent onboarding context copied from the kit
- `backlog/`: deferred follow-ups and open action items
- `runs/`: active and archived run state/history
- `outputs/`: plans, validations, reviews, handoffs
