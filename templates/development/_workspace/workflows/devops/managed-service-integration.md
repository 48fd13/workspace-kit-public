# Managed Cloud Service Integration Review

Use when integrating or changing managed services such as queues, object storage, databases, caches, secrets managers, event buses, schedulers, email, or search services.

Use `_workspace/runbooks/managed-service-integration.md`.

Check auth, network access, data ownership, retries, quotas, limits, lifecycle, local/dev parity, observability, cost, and fallback/rollback behavior.
