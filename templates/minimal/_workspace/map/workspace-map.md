# Workspace Map

| Path | Purpose |
|---|---|
| `AGENTS.md` | Local operating policy |
| `repo/` | Project code, repository, manifests, and application files |
| `repos/<name>/` | Multiple repositories when needed |
| `_workspace/map/` | Routing, naming, workspace map |
| `_workspace/workflows/` | Workflows and checklists |
| `_workspace/pipelines/` | Multi-stage pipelines with review between stages |
| `_workspace/context/reference/` | Template-provided reference context |
| `_workspace/context/project/` | Reviewed stable project reference |
| `_workspace/backlog/` | Unresolved questions, deferred follow-ups, risks, and open action items |
| `_workspace/runs/` | Active, finished, and archived standalone task/workflow/pipeline state |
| `_workspace/outputs/` | Durable artifacts promoted when runs finish |
