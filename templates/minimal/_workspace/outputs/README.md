# Outputs

Use this folder for durable artifacts promoted when a run is explicitly finished. Drafts stay under the active run until then.

Every finished run records at least one output path in `RUN.md` frontmatter. When changed target files are the main result, create a concise completion handoff.

Common folders:

- `plans/`
- `reviews/`
- `validations/`
- `handoffs/`
- `decisions/`
- `retrospectives/`
