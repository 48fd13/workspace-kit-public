# Naming Conventions

Use durable artifact names:

```text
YYYY-MM-DD-scope-artifact-vNN.md
```

Common locations:

```text
_workspace/outputs/handoffs/YYYY-MM-DD-topic-handoff-vNN.md
_workspace/outputs/plans/YYYY-MM-DD-topic-plan-vNN.md
_workspace/outputs/reviews/YYYY-MM-DD-topic-review-vNN.md
_workspace/outputs/validations/YYYY-MM-DD-topic-validation-vNN.md
_workspace/outputs/maps/YYYY-MM-DD-scope-map-type-vNN.md
_workspace/outputs/change-packets/YYYY-MM-DD-scope-change-packet-vNN.md
_workspace/outputs/incident-notes/YYYY-MM-DD-incident-or-alert-vNN.md
_workspace/outputs/runbooks/YYYY-MM-DD-topic-runbook-vNN.md
_workspace/outputs/decisions/YYYY-MM-DD-topic-decision-vNN.md
_workspace/backlog/items/YYYY-MM-DD-short-slug.md
_workspace/runs/active/YYYY-MM-DD-topic-slug/
_workspace/runs/archive/YYYY-MM-DD-topic-slug/
```

Map examples:

```text
2026-06-12-payments-api-service-map-v01.md
2026-06-12-prod-environment-map-v01.md
2026-06-12-platform-ownership-map-v01.md
2026-06-12-payments-dependency-map-v01.md
```
