# Handoff Workflow

Use when the user asks to hand off, preserve state, or continue later.

## Process

1. Summarize goal, current state, decisions, files changed, validation, risks, and next actions.
2. Include relevant artifact paths.
3. Save to `_workspace/outputs/handoffs/YYYY-MM-DD-topic-handoff-vNN.md`.
4. Reply with path, one-line summary, and next action.
